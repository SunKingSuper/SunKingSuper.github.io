怎样用Markdown发推送

  • SKS
  • 0 Minutes
  • November 8, 2018

前言

微信公众号推送的排版一直是个老大难的问题，像微信官方提供的后台排版工具又过于简陋。
于是乎，又有一批需要注册或者收费的排版网站专门帮助排版。然而，对于像我这种不爱折腾的人来说，
平时发的都是技术文章（3个月出一篇），只要有个样式好看就行。
由于最近在做项目的总结推送，对推送排版难的问题感到很头疼，于是搜了搜，发现了一款 FireFox 的插件——
Markdown Here 这款神器，能够解决大多数想用 Markdown 排版的应用场景。

介绍

Markdown Here 这款神器， 是一个支持 FireFox/Chrome/Safari 的浏览器插件，当你在浏览器应用商店下载好并安装后，
就会弹出一个页面来设置初始设定。然后你就可以将你写好的 Markdown 文本，直接复制到相应位置，然后全部选中，再按默认热键
Ctrl+Shift+M ，就可以把你原来的文本按照 Markdown 语法解析进行排版。
