Markdown语法总结（Github flavoried）

Markdown 是一种很方便的格式控制语言，能够快速排版好我们想要发表的文章，由于其简单的语法，备受程序员喜爱。

我认为学习 Ｍarkdown 语法的方法就是要记住 Markdown 所有的格式以及其语法，要注意 Markdown 不存在换行的概念，更多时候 Markdown 会忽略掉空白符（回车、空格），所以换行、分段更多靠一个空行隔开。

我是根据 Github Guides
的内容来写下面练习，大家通过练习和答案，希望能帮助到大家记住 Markdown 用法。
